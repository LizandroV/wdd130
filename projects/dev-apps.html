Applications

In an increasingly technology-driven world, automation has become a vital component for business success. My applications have been developed with the goal of simplifying repetitive tasks and freeing up valuable resources within an organization. These solutions are designed to ensure maximum operational efficiency while reducing errors and improving the quality of results.

Each application is designed to optimize efficiency and maximize productivity. From project management tools to data analysis solutions, each application offers specific functionality to overcome common business challenges.





FOTI-UPDATE

Created to automate the update of a VPN application, the network update was applied saving time and resources

STEPS

  • Save a Backup file
  • Check if the version is higher than 7.0
  • uninstall outdated app
  • Install new App version
  • Restore Backup file

BACKUP-REPORT

Created to show the status of the backups on servers, this application sends a daily email indicating the status of the backups

STEPS

  • Check files of backup by date
  • If there is no file of the day there was an error
  • Update info to the BBDD
  • Create a report in PowerBI
  • Send report by email daily
  • Save report as image into a folder

ChatBOT

Created to give quick technical support responses, which saves time for the user and saves company resources.

STEPS

  • Wait for messages
  • If there is a message the app will send a message
  • User can check if his problem is listed
  • If the problem is not resolved, a technical staff is referred
  • A request is created in an excel database
  • when the problem is solved, the cycle is terminated
